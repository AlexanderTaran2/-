Ответы на вопросы для самопроверки:
1. Инструменты Figma: Инспектор, Measure tool, Color Picker, Text tool, Layers panel, Export panel.
2. Организация CSS-переменных: Группировка по цветам, типографике, отступам, размерам в :root с семантичными названиями.
3. Критичные метрики: Размеры контейнеров, межстрочный интервал, межбуквенный интервал, отступы (padding/margin), выравнивание, размеры шрифтов.
4. Определение сетки: Измерить отступы между колонками, количество колонок, выравнивание, адаптивное поведение → перенос в CSS Grid/Flexbox.
5. Сложности типографики: Доступность шрифтов, различия рендеринга, line-height, адаптивность, производительность загрузки.
6. Адаптивность без макета: Mobile-first подход, анализ компонентов, относительные единицы, fluid typography, консультация с дизайнером.
7. Инструменты браузера: Overlay макета, Measure tool, Color picker, Font inspector, Box model viewer, Lighthouse.
8. Экспорт изображений: PNG (прозрачность), JPG (фото), SVG (иконки), WebP (современный); настройка 1x/2x/3x; оптимизация качества.
9. Отсутствующие шрифты: Поиск альтернатив (Google Fonts), веб-шрифты (@font-face), системные шрифты, переговоры о замене.
10. Методологии соответствия: BEM, CSS-переменные, компонентный подход, дизайн-токены, документация, регулярные проверки.
